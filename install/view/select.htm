FoundPHP FrameWork选择数据库
{FoundPHP_Ver}
{support_database['mysql']}

MySQL 数据库体积小、速度快、开放源代码，中小型网站的开发都选择 MySQL 作为网站数据库。1995 年发布

{support_database['mariadb']}

MariaDB基于事务的Maria存储引擎，兼容MySQL语法,拥有更多全新特性，中小型网站的选择。2012 年发布

{support_database['sqlite']}

SQLite 就是一个文件轻量级嵌入式数据库，实现了无服务器、事务性的SQL数据库引擎。
2000 年发布

SQL Server

SQL Server 基于windows拥有易用性、负载高、可伸缩性及可靠性，与各种win软件完美结合。
1989 年发布

{support_database['pgsql']}

PostgreSQL是全功能的自由软件数据库，唯一支持事务、子查询、多版本并行控制系统的数据库。1996 年发布

{support_database['oracle']}

Oracle 可移植性好、使用方便、功能强，效率高、可靠性好、超大吞吐量、分布式的数据库。1979 年发布

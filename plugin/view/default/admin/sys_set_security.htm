前端 GET 数据入口


  开启后将对GET数据入口限制
  逗号分隔接收数据入口，没有入口则不接收
入口 介绍
m model 操作模块
a action 动作行为
t to 跳转
o operation 操作
g go 成功失败状态
s search 搜索
k keys 搜索键值
p page 分页id
id id 产品序列
cid cid 分类序列
token token 接收实现远程登陆

安防设置 用于防止攻击功能


  对所有操作记录
  记录前后台每个页面地址以及数据库语句，访问量超大的时候会影响效率。
  开启后可以启动自动建立页面，关闭增加安全性
  在页面的下方显示调用模板与数据库语句信息

跨域设置 允许别的域名访问本站


  允许ajax或其他应用对网站访问

  每一行代表一个可以允许访问的域名，例如：www.FoundPHP.com,如果想全部接受则输入*
  针对授权模块开发域名授权，例如：api，多个以逗号分隔
